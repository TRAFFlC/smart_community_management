# 智慧社区管理平台 —— 功能模块清单 · 用例表 · 权限矩阵

## 1. 文档目标

本文档是 `spec.md` 的细化延伸，将系统规格转化为：

- **功能模块清单**：逐级拆解每个模块的子功能、数据实体、核心操作。
- **用例表**：以角色为中心描述典型使用场景，包含前置条件、主流程、异常流程。
- **权限矩阵**：以"角色 × 功能"方式定义每个角色对每个功能的操作权限（查看/新增/编辑/删除/审批/导出）。

作为后续数据库设计、接口设计和前端页面开发的直接依据。

## 3. 用例表

### 3.1 UC01 — 居民报事报修

| 项目 | 说明 |
| --- | --- |
| **用例编号** | UC01 |
| **用例名称** | 居民报事报修 |
| **参与角色** | 居民(户主/家庭成员/租户)、物业客服、物业管家、维修人员、物业经理 |
| **前置条件** | 居民已注册并完成实名认证，已绑定房屋 |
| **主流程** | ① 居民选择"报事报修"，填写位置（自动关联房屋）、问题类型、描述、上传图片 → ② 系统创建工单，状态="待受理"，根据小区和问题类型自动分配至物业客服 → ③ 物业客服查看工单，确认受理，派发给对应维修/保洁/安保人员 → ④ 执行人员接收工单，现场处理，上传处理照片和文字记录 → ⑤ 处理完成，工单状态="已完成"，系统通知居民 → ⑥ 居民查看处理结果，进行满意度评价(1-5星+文字) → ⑦ 工单归档，数据进入统计报表 |
| **异常流程** | A. 物业48h未受理 → 系统自动预警通知物业经理；B. 执行人员无法处理 → 升级给物业经理或转社区事件；C. 居民评价不满意 → 可发起"重新处理"请求 → 工单重新打开 |
| **后置条件** | 工单归档，统计数据更新 |
| **业务规则** | 1. 报修类型：水电维修、公共设施、环境卫生、安全秩序、其他；2. SLA：普通问题48h、紧急问题4h、重大安全问题2h |

### 3.2 UC02 — 社区治理事件处理

| 项目 | 说明 |
| --- | --- |
| **用例编号** | UC02 |
| **用例名称** | 社区治理事件全流程处理 |
| **参与角色** | 网格员/居民/物业(上报者)、社区工作人员(审核)、承办人、街道工作人员(督办)、联动单位 |
| **前置条件** | 上报者已登录且具备上报权限 |
| **主流程** | ① 上报者填写事件信息(类别、位置、描述、图片、定位) → ② 系统创建事件，状态="待审核" → ③ 社区工作人员审核事件，确认类别、优先级、责任方 → ④ 分派给具体承办人或联动单位 → ⑤ 承办人处理事件，上传处理过程和结果 → ⑥ 处理完成，社区工作人员审核结果 → ⑦ 对上报者/相关居民进行回访 → ⑧ 事件归档，纳入考核统计 |
| **异常流程** | A. 审核不通过 → 退回上报者补充信息；B. 超时未处理 → 系统自动预警 → 督办人员介入；C. 需联动处置 → 发起联动请求 → 协同单位响应 → 联合处理；D. 事件升级 → 转上级部门处理 |
| **后置条件** | 事件归档，考核数据更新 |
| **业务规则** | 事件类别：民生服务、环境卫生、设施安全、邻里纠纷、特殊人群帮扶、市容秩序、突发预警；优先级：一般/重要/紧急/特急；SLA按类别和优先级配置 |

### 3.3 UC03 — 业委会监督与投票

| 项目 | 说明 |
| --- | --- |
| **用例编号** | UC03 |
| **用例名称** | 业委会监督与投票表决 |
| **参与角色** | 业委会主任、业委会成员、居民(业主)、物业 |
| **前置条件** | 业委会组织已在系统中注册，成员已绑定 |
| **主流程** | ① 物业发布公共收益公示或重大事项报告 → ② 业委会成员查看并提出意见 → ③ 业委会主任决定是否需要投票 → ④ 发起投票议题(设定投票时间、参与范围、选项) → ⑤ 系统通知符合条件的业主 → ⑥ 业主在线投票 → ⑦ 投票结束，系统自动统计结果 → ⑧ 结果公示并归档 |

## 6. 补充说明

### 6.1 角色权限叠加规则

- 一个用户可拥有多个角色，权限取**并集**。
- 当数据权限和功能权限冲突时，以**最高权限角色**为准。
- 平台级权限不受数据范围限制。

### 6.2 权限变更生效

- 角色/权限变更后，用户**下次请求**即生效（无需重新登录）。
- 重大权限变更（如角色删除）需强制用户重新登录。

### 6.3 审计要求

- 所有权限变更操作必须记录审计日志。
- 敏感数据访问（特殊群体、身份证等）必须记录访问日志。
- 日志保留期限不少于 180 天。
