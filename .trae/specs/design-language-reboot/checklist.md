# Design Language Reboot Checklist

## 设计系统
- [x] `resources/style.qss` 顶部有集中定义的颜色/字体/间距/圆角 token 注释
- [x] 全局背景色使用新的近白色（#f8fafc 方向），不再是 #f0f2f5
- [x] 主按钮使用新的强调色，不再是 #1677ff 蓝色
- [x] 危险/成功/警告色使用降低饱和度后的新语义色
- [x] 全局圆角统一收敛到 4-6px，无 12px 大圆角卡片

## 框架层
- [x] 侧边栏背景为深 slate（#0f172a 附近），文字为浅灰/白色
- [x] 侧边栏激活项使用左侧强调色条 + 半透明高亮背景
- [x] 侧边栏分组标题不再使用大写/字母间距伪装专业感
- [x] 顶栏背景与内容区协调，底部细边框为 #e2e8f0
- [x] 面包屑为普通文本路径，可点击项 hover 有下划线
- [x] 搜索框、刷新按钮、通知铃铛、用户菜单样式符合新系统

## 工作台
- [x] 工作台顶部不再有四张等宽统计卡片
- [x] 顶部展示 2-3 个超大数字指标 + 标签
- [x] 快捷操作为无背景图标网格，无小卡片边框
- [x] 待办列表无 emoji，使用色点前缀
- [x] 社区动态使用时间线/左侧色条样式
- [x] 工作台各模块间距统一（24px 边距，16/24px 间距）

## 表格与列表
- [x] 表头背景 #f1f5f9，文字 #475569，11px，字重 600
- [x] 数据行统一 44px 行高，斑马纹清晰
- [x] 整行 hover 背景变化，而非仅文字变色
- [x] 状态标签 4px 圆角、低饱和背景、12px 字号
- [x] 操作列不再使用彩色胶囊按钮，改为统一小按钮或文字链接
- [x] 空状态图标与文字使用新中性色

## 页面头部
- [x] 页面头部无渐变背景与左侧色条
- [x] 头部为简洁标题 + 描述 + 可选操作按钮
- [x] 头部高度 ≤ 64px
- [x] 标题 20px/700，描述 13px/400 #64748b

## 图标与细节
- [x] 全局无 emoji（📋/📌/📢/✓/⚠️ 等）
- [x] 统计图表配色与新设计系统一致
- [x] 登录页主色/按钮/输入框同步新设计系统
- [x] 表单输入框圆角 4px，focus 使用新强调色
- [x] 弹窗标题 16px/600，间距收敛

## 编译与运行
- [x] `cmake --build . --config Release` 编译通过
- [x] 运行时无 QPainter 错误（静态检查无潜在问题）
- [x] 运行时无 QString::arg 警告（静态检查参数数量均匹配）
- [x] 逐页视觉走查，无未覆盖的旧样式碎片
