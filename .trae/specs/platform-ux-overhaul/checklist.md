# Platform UX Overhaul Checklist

## Phase 1: 安全与基础交互
- [x] 全局 SQL 注入修复：所有搜索/筛选查询使用参数绑定，无字符串拼接
- [x] 登录页不再预填 admin/admin123，改为占位符提示
- [x] 通知铃铛显示未读数红色角标，0时隐藏，>99显示"99+"
- [x] 面包屑各级别可点击导航到对应页面
- [x] 顶栏有刷新按钮，点击后刷新当前页面数据
- [x] 刷新按钮有旋转动画反馈

## Phase 2: 表格交互统一
- [x] 所有状态标签使用统一的圆角背景样式（createTagItem）
- [x] 表格操作列为胶囊式 QPushButton（带背景色+hover效果），非纯文字
- [x] 所有表格为空时显示空状态提示（图标+文字+引导）
- [x] 工单/事件/投诉/民意/督办/志愿的操作前有确认弹窗
- [x] 确认弹窗文案具体化（如"确认派单给张三？"）
- [x] 所有写入操作成功后显示 Toast 成功提示（2秒自动消失）
- [x] 所有数据表格添加分页控件（每页20/50/100条切换 + 页码导航 + 跳转）
- [x] 每张表格底部显示"共 N 条记录"（集成在分页控件中）

- [x] 用户管理：可新增用户（含角色分配）
- [x] 用户管理：可编辑用户信息
- [x] 用户管理：可禁用/启用用户
- [x] 用户管理：可重置密码
- [x] 角色管理：可新增角色（含菜单权限分配）
- [x] 角色管理：可编辑角色权限
- [x] 菜单管理：可新增/编辑/删除菜单项

- [x] 字典管理：可编辑字典项
