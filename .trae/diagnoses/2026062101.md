→ 建议将 alpha 提升到 40-60，让建筑轮廓在常规显示器上可见。



\*\*2. \[表格表头] 深色表头与浅色内容对比过强\*\*



\[style.qss](file:///d:/smart\_community\_management/resources/style.qss#L140-L157) 表头使用 `#141413`（近黑）背景 + `#FAF9F6`（暖白）文字，而表格内容是白色背景。这种极端的明暗切换在小表格中尚可，但在大表格（20+行）中会造成视觉疲劳 — 用户的眼睛在深色表头和浅色内容之间反复适应。



→ 建议表头改为 `#F5F2EB` 背景 + `#141413` 文字 + 底部 2px `#B45309` 线条，更温和且保持权威感。



\*\*3. \[全局] 2px 圆角过于锐利\*\*



全局圆角统一为 2px，在高分辨率显示器上几乎等于直角。虽然设计意图是"克制"，但现代 UI 趋势中 4-6px 的圆角既克制又不会显得生硬。当前的 2px 在卡片、输入框、按钮上看起来像是"忘了设圆角"。
