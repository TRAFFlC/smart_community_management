# 智慧社区管理平台

基于 C++ Qt 6 的桌面端智慧社区管理系统，涵盖小区管理、社区治理、社区服务、统计分析、系统管理五大模块。

## 技术栈

| 组件 | 技术 |
|------|------|
| 语言 | C++17 |
| GUI 框架 | Qt 6 Widgets |
| 数据库 | SQLite (本地) |
| 图表 | Qt Charts |
| 构建 | CMake 3.16+ |
| 编译器 | MinGW GCC 14.2.0 |

## 功能模块

### 工作台
- 统计概览卡片（工单、事件、居民、小区）含趋势指示器（↑↓ 百分比）
- 快捷操作入口、待办事项列表
- 社区动态 & 工单状态分布图表
- 数据加载状态指示条

### 基础档案
- 组织管理、楼栋房屋、居民管理、车辆管理、设施管理、网格管理、特殊群体

### 物业管理
- 报事报修（全生命周期：受理→派单→处理→完成→评价）
- 投诉建议、公告通知、物业巡检、访客管理、民意收集
- 业委会议题投票、停车管理、物业缴费、公共收益

### 社区治理
- 网格事件、社区巡查、重点人群关怀、督办管理、考核管理

### 社区服务
- 志愿服务（活动组织与报名）、便民服务、就业服务
- AI 智能问答（知识库匹配 + 关键词模糊搜索）

### 统计分析
- 工单统计（月度趋势折线图、优先级分布柱状图、类型分布饼图）
- 事件统计（超时分析、状态停留时长柱状图、类别分布饼图）
- 服务统计（类型分布柱状图、满意度评分分布图、状态分布饼图）
- 综合数据看板（多维度可视化）

### 系统管理
- 用户管理、角色管理（RBAC 权限模型）、菜单管理、字典管理、操作日志

## 构建与运行

### 前置条件

- Qt 6.5+ (需包含 Charts 模块)
- CMake 3.16+
- MinGW GCC 14+ 或 MSVC 2022+

### 构建步骤

```bash
# 配置
cmake -B build -G "MinGW Makefiles" -DCMAKE_BUILD_TYPE=Release

# 编译
cmake --build build

# 运行
./build/SmartCommunity.exe
```

> **注意**：首次运行时，若数据库为空会自动创建 58 张表并填充演示数据（17 个测试账号、小区档案、工单、事件等）。

## 权限模型

系统采用 RBAC（基于角色的访问控制）模型：

- **菜单权限**：不同角色看到不同的侧边栏菜单（如居民看不到系统管理模块）
- **数据范围**：支持平台/街道/社区/小区/楼栋/个人/协作 7 级数据权限
- **操作权限**：基于角色的操作按钮控制（如居民只能评价工单，不能受理/派单）

## 设计规范

- **禁止硬编码数据**：所有页面数据来源于数据库查询
- **数据库表命名**：`sys_` 系统管理、`cm_` 社区档案、`wo_` 工单、`ge_` 治理事件、`sv_` 社区服务、`nt_` 通知、`pm_` 物业管理、`oc_` 业委会等
- **审计字段**：每张表包含 `create_by`、`create_time`、`update_by`、`update_time`、`del_flag`
